# ✈️ Travel Booking App UI

## 📌 Project Overview
This project is a Travel Booking App UI designed in Figma. It provides a simple and modern interface for users to search flights, view destinations, make payments, and complete bookings.

---

## 🎯 Objective
The main objective of this project is to design a clean, user-friendly, and attractive travel booking application interface using Figma while following modern UI/UX design principles.

---

## ✨ Features
- Beautiful Splash Screen
- Home Screen with Travel Destinations
- Flight Search Screen
- Booking Summary Screen
- Payment Screen
- Booking Success Screen
- Modern and Clean UI Design
- Easy Navigation

---

## 🛠️ Tools Used
- Figma
- UI/UX Design Principles
